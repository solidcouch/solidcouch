# Instances of SolidCouch

You can join one of these communities:

- [Tired Bike](https://tired.bike) - Community for touring cyclists and other slow travellers.

- [Solid Folks](https://folks.solidcouch.org) - An instance for folks involved in Solid Project.

- [SolidCouch Test instance](https://app.solidcouch.org) - Testing community running the latest SolidCouch software. It may change or disappear without notice.

- Start your own!
