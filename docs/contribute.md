## Add yourself to Contributors list

This project follows the [all-contributors](https://github.com/all-contributors/all-contributors) specification.

If you have contributed to SolidCouch in any capacity, you deserve a place in the [Contributors](https://github.com/solidcouch/solidcouch/#contributors-) list.

To be added, either:

1. Reach out to the maintainers via Matrix, or
2. Run the following in the [main SolidCouch repository](https://github.com/solidcouch/solidcouch) and open a PR:
   ```bash
   yarn contributors:add <github-username> <contribution-keys>
   yarn contributors:generate
   ```
   See [contribution keys reference](https://allcontributors.org/emoji-key/).

We try to keep the list up to date but may miss some contributions. Don't hesitate to ask for credit you deserve.
